A React web application for staking SUI tokens to receive afSUI using the Aftermath SDK.

## Setup & Run Instructions

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Open `http://localhost:8080` in your browser.

### Production Build

```bash
npm run build
```

Output: `dist/` directory

## Technologies

- **React** + **TypeScript**
- **@mysten/dapp-kit** - Sui wallet integration
- **@mysten/sui** - Sui blockchain client
- **aftermath-ts-sdk** - Staking SDK
- **Tailwind CSS** - Styling
- **Webpack** - Build tool
